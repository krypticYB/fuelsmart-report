Active Support Items
Tweaks and refinements currently in progress
Requisition Approval Hierarchy In Progress

Refining the approval chain on the requisition form to match the Chairman's preferred flow: the Supervisor forwards the request (without approving it), it then moves to the Manager, then the Accountant, and finally to the Chairman for final sign-off. This is a configuration update to the existing form, not a rebuild.

✓ Being configured in Zoho People approval settings. Target: complete before end of April.

Reports Module — Manager Adoption In Progress

The manager has expressed a preference for entering figures with comma thousand-separators alongside decimals (e.g. 1,234.56). Zoho does not support both in the same input field by design. The agreed approach is for the manager to enter clean numeric values, which the system then displays in a formatted way on reports.

✓ Manager is being guided through the correct entry method during support sessions.

Attendance — Awaiting Hardware Blocked Externally

The Zoho People attendance module is fully configured on the platform side. It cannot go live until a physical attendance device (biometric or otherwise) is installed at the office. This is outside the consultant's scope and depends on the office taking action to procure and install the hardware.

✓ No action required on the Zoho side. Will proceed immediately once hardware is in place.
